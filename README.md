## 🚀 Sobre este Repositorio

Este repositorio contiene los trabajos, ejercicios y el proyecto final integrador del curso de **Machine Learning** dictado por **Talento Tech** en la Ciudad de Buenos Aires. 
El trayecto formativo se enfoca en el ciclo completo de un proyecto de aprendizaje automático, desde la preparación de datos hasta la implementación de modelos predictivos.

### 🎓 Certificación
Al finalizar, este trayecto otorga una **microcredencial oficial** del Ministerio de Educación de la Ciudad de Buenos Aires.

---

## 📚 Contenidos del Curso

El programa se divide en cuatro módulos principales que cubren tanto el aprendizaje supervisado como el no supervisado:

**Módulo 1:** Introducción y revisión de herramientas (Pandas, Matplotlib, Seaborn, Plotly).
**Módulo 2:** Fundamentos de ML y preprocesamiento avanzado (Feature Engineering, Normalización).
**Módulo 3:** Modelos y técnicas de aprendizaje (Regresión, Clasificación, Clustering y Pipelines).
**Módulo 4:** Proyecto guiado y Proyecto Final Integrador.

### 🛠️ Tecnologías y Herramientas
**Entornos:** Google Colab, Kaggle.
**Librerías:** Scikit-learn, Pandas, NumPy, SciPy.
**Modelos:** Regresión Lineal/Logística, KNN, Árboles de Decisión, Random Forest y K-Means.

---

## 📈 Métricas y Evaluación

Para garantizar la calidad de los modelos desarrollados, se aplican las siguientes métricas

| Tipo de Modelo | Métricas Utilizadas |
| :--- | :--- |
| **Regresión** |MAE, MSE, RMSE, $R^{2}$ |
| **Clasificación** |Precisión, Recall, F1-score, Matriz de Confusión |
| **Clustering** |Inercia, Silhouette Score |

---

## 🏁 Proyecto Final Integrador

El repositorio culmina con un **Proyecto Final Integrador** que consiste en el desarrollo completo de un modelo aplicado a un caso real. Este proyecto incluye:
1.  Selección y exploración de un dataset real.
2.  Preprocesamiento e ingeniería de características.
3.  Implementación y optimización de modelos mediante **GridSearchCV**.
4.  Evaluación de resultados y comunicación de hallazgos.
