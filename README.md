## 🚀 Sobre este Repositorio

[cite_start]Este repositorio contiene los trabajos, ejercicios y el proyecto final integrador del curso de **Machine Learning** dictado por **Talento Tech** en la Ciudad de Buenos Aires[cite: 3, 1, 5]. [cite_start]El trayecto formativo se enfoca en el ciclo completo de un proyecto de aprendizaje automático, desde la preparación de datos hasta la implementación de modelos predictivos[cite: 32].

### 🎓 Certificación
[cite_start]Al finalizar, este trayecto otorga una **microcredencial oficial** del Ministerio de Educación de la Ciudad de Buenos Aires[cite: 15, 16].

---

## 📚 Contenidos del Curso

[cite_start]El programa se divide en cuatro módulos principales que cubren tanto el aprendizaje supervisado como el no supervisado[cite: 73, 52]:

* [cite_start]**Módulo 1:** Introducción y revisión de herramientas (Pandas, Matplotlib, Seaborn, Plotly)[cite: 74, 102].
* [cite_start]**Módulo 2:** Fundamentos de ML y preprocesamiento avanzado (Feature Engineering, Normalización)[cite: 77, 111].
* [cite_start]**Módulo 3:** Modelos y técnicas de aprendizaje (Regresión, Clasificación, Clustering y Pipelines)[cite: 81, 140, 141].
* [cite_start]**Módulo 4:** Proyecto guiado y Proyecto Final Integrador[cite: 92].

### 🛠️ Tecnologías y Herramientas
* [cite_start]**Entornos:** Google Colab, Kaggle[cite: 26, 33].
* [cite_start]**Librerías:** Scikit-learn, Pandas, NumPy, SciPy[cite: 26, 51].
* [cite_start]**Modelos:** Regresión Lineal/Logística, KNN, Árboles de Decisión, Random Forest y K-Means[cite: 54, 55].

## 📈 Métricas y Evaluación

[cite_start]Para garantizar la calidad de los modelos desarrollados, se aplican las siguientes métricas[cite: 46, 68]:

| Tipo de Modelo | Métricas Utilizadas |
| :--- | :--- |
| **Regresión** | [cite_start]MAE, MSE, RMSE, $R^{2}$ [cite: 46, 123] |
| **Clasificación** | [cite_start]Precisión, Recall, F1-score, Matriz de Confusión [cite: 46, 129] |
| **Clustering** | [cite_start]Inercia, Silhouette Score [cite: 46, 138] |

## 🏁 Proyecto Final Integrador

[cite_start]El repositorio culmina con un **Proyecto Final Integrador** que consiste en el desarrollo completo de un modelo aplicado a un caso real[cite: 150, 153]. Este proyecto incluye:
1.  [cite_start]Selección y exploración de un dataset real[cite: 151].
2.  [cite_start]Preprocesamiento e ingeniería de características[cite: 151].
3.  [cite_start]Implementación y optimización de modelos mediante **GridSearchCV**[cite: 151, 69].
4.  [cite_start]Evaluación de resultados y comunicación de hallazgos[cite: 151].
